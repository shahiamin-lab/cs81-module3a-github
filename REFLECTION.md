# Reflection

## What did you learn about using Git and GitHub?
I learned how Git tracks changes through commits and how GitHub stores and shares code online.

## What was most confusing or frustrating?
Understanding when to commit and how staging works was initially confusing.

## Why do developers use Git?
Git allows developers to track changes, collaborate, and safely manage large projects.

## How might this help with larger projects?
It helps organize work, recover older versions, and work with teams efficiently.
